# Data-Analysis-on-CSV-Files
Data Analysis Project Structure
​For "Data Analysis on CSV Files", these are the recommended file names and their purposes.
​Deliverable Files
​1. sales_data.csv

​Purpose: Contains the raw sales dataset.
​Columns: Date, Product, Region, Quantity, UnitPrice.

​2. sales_analysis.ipynb (or .py)

​Purpose: The Jupyter Notebook or Python script containing the analysis code.

​Key Operations:

​Loading sales_data.csv.
​Grouping data by Product and Region.
​Calculating sums (Total Sales).
​Generating visualization plots.



​3. sales_charts.png

​Purpose: The output image file containing the generated charts (Bar chart and Pie chart).
​Note: This is generated automatically by the script.

​How to Run
1.
Ensure sales data.csv and sales_analysis. py are in the same folder.

2. Run the script using Python:

   ```bash python sales_analysis.py

3. Check the folder for the new
    sales_charts.png file.
